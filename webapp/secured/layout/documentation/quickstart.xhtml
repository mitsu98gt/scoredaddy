Quickstart Guide - Use this guide as a quick reference for the applications workflow.

  1. Add Tournament - A tournament is a single representation of the entire GSSF event that includes all three (3) competitions. For example, the name of your tournament could be named "GSSF Summer Tournament".
  2. Add Competition - A tournament event is represented by competitions. GSSF Indoor tournaments will be composed of three (3) competitions. For example, the name of your competition could be "GSSF Summer Shoot 1", "GSSF Summer Shoot 2", and "GSSF Summer Shoot 3" respectively. Only create competitions when you are ready to register competitors and scoring.
  3. Add Competitor - Add NEW competitors. You will only have to this once for each new competitor to your events.
  4. Register Competitor - Register your competitor to the competition. You may use the drop down menu OR type the first few characters of the competitors name for a lookup functionality.
  5. Scoring->GSSF Scoresheet - Use this page to input, download GSSF scoresheets, and save competitor score entry.
  6. Scoring->Update Scores - Use this page if you want to delete a competitors score entry that was saved incorrectly. Select the competition and division of the entry you want to delete. Then select the radio button of that entry to highlight the row, and click on the Trash Can ICON.
  7. Results->View GSSF Scores - Scores can be viewed in Real Time as they are being inputted for each competition. The "Averaged Scores" button will only be enabled once you have at leaset TWO(2) competitions created with scores. Once this button is enabled, you may view the current tournament averaged competitor scores and standings.
